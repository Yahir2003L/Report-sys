<h2>Asegurese de primero bajar el repositorio</h2>

#Pasos después de clonar el repositorio
1. Instalar dependencias:
   <p>En la terminal del proyecto</p>

    ```
      npm install
     ```
2. Crear una base de datos con el nombre "sys_reports".
3. Importar el archivo .sql del repositorio a la base de datos previamente creada.
4. Copear el archivo .env.example a un arichivo .env. Si tiene una cuenta, favor de poner su username y password en dichos campos en el archivo .env
5. Correr el proyecto con el siguiente comando:

    ```
      npm run dev
     ```
